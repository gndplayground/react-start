My personal react starter kit. 
Fork and edit original repo [react-hot-boilerplate](https://github.com/gaearon/react-hot-boilerplate/tree/next)

## Stacks

* React
* Redux
* Redux saga
* React Router v4

## Commands

### ESLint config with `airbnb`
`yarn run lint`

### Dev with hot loader
`yarn start`

### Build bundle
`yarn build`
